# 📊 E-Commerce Sales & Profit Performance Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive Power BI dashboard analyzing e-commerce sales performance, profitability trends, customer contribution, and product-level insights.

The dashboard is designed to provide a clear executive-level summary and support data-driven business decisions.

---

## 🎯 Objective
To analyze overall sales performance, track profit trends, and evaluate customer and product-level contribution using structured KPIs and interactive visualizations.

---

## 📈 Key KPIs

- **Total Sales:** 2M  
- **Total Profit:** 37K  
- **Total Quantity Sold:** 5615  
- **Average Order Value (AOV):** 121K  

These KPIs provide a high-level overview of business performance.

---

## 📊 Dashboard Insights

- Achieved 2M in total sales with 37K overall profit.
- Monthly profit analysis highlights performance fluctuations, including low or negative profit periods.
- Printers and Bookcases emerged as high-profit sub-categories.
- A small group of customers significantly contributes to overall revenue.
- City-wise sales distribution helps identify strong-performing regions.
- Category and sub-category analysis supports product-level decision-making.

---

## 🛠 Tools & Technologies Used

- Power BI  
- DAX (SUM, DIVIDE, Basic Measures)  
- Data Modeling  
- KPI Design  
- Interactive Data Visualization  

---

## 🚀 Business Value

This dashboard helps businesses:

- Monitor sales and profit performance
- Identify high-performing products and regions
- Focus on high-value customers
- Improve low-profit periods
- Support strategic, data-driven decisions
